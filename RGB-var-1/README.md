## Описание системы

[**Ссылка на проект**](https://github.com/nurdashymside/Smart-Lights)

- **Контроллер**
    - ESP32
- **Силовой транзистор**
    - IRF3205
    - IRF3704ZPBF
    - IRLB8743PBF
    - RL2203NPBF
    - IRLB8748PBF
    - IRL8113PBF
    - IRL3803PBF
    - IRLB3813PBF
- **Светодиодная лента**
    - Светодиодная лента (прототип)
    - RGB лента (возможность управлять цветом)
    - WS2812B (возможность управлять каждым чипом светодиода)

- **RGB лента**
